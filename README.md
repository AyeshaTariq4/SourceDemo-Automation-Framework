# SauceDemo Automation Framework

## Project Description
This project automates the SauceDemo website using Selenium WebDriver with Python.

## Features
- Browser Launch
- Login Automation
- Add Product to Cart
- Open Shopping Cart
- Checkout Process
- Order Completion
- Page Object Model (POM)
- Headless Chrome Support
- Screenshot Capture

## Technologies Used
- Python
- Selenium Webdriver
- WebDriver Manager
- ChromeDriver
- Page Object Model (POM)
- Git
- GitHub
- GitHub Actions
- VS Code
