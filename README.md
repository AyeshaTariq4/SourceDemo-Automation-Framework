# SauceDemo Automation Framework

## Project Description
This project automates the SauceDemo website using Selenium WebDriver with Python.

## Features
- Browser Launch
- Login Automation
- Add Product to Cart
- Open Shopping Cart
- Checkout Process
- Order Completion
- Page Object Model (POM)

## Technologies Used
- Python
- Selenium
- WebDriver Manager
- ChromeDriver
- VS Code

## Project Structure

SauceDemo_Automation/
│
├── pages
├── tests
├── utilities
├── reports
├── screenshots
├── requirements.txt
└── README.md
